# Guangdong Mahjong Learning Website

## 功能特点

- 🎨 **多邻国风格设计** - 鲜艳的色彩、圆润的卡片、友好的界面
- 🎯 **交互式学习** - 点击、悬停、展开等丰富的交互效果
- 📱 **响应式设计** - 支持桌面和移动设备
- ✨ **动画效果** - 流畅的过渡动画和视觉反馈
- 📊 **进度追踪** - 顶部进度条显示阅读进度
- 🎮 **游戏化元素** - 卡片式布局、步骤式学习

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器运行（推荐）：
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 使用 Node.js
   npx http-server
   ```
3. 在浏览器中访问 `http://localhost:8000`

## 主要内容

1. **Tile Set** - 麻将牌介绍（136张牌）
   - 花色牌（条、万、筒）
   - 字牌（风牌、箭牌）

2. **Game Goal** - 游戏目标
   - 胡牌组合示例
   - 红中万能牌说明

3. **Game Setup** - 游戏设置
   - 座位安排
   - 洗牌和建墙
   - 初始摸牌

4. **Gameplay** - 游戏玩法
   - 摸牌、出牌
   - 碰牌（Pung）
   - 杠牌（Kong）的三种类型

5. **Tips** - 新手提示
   - 实用的游戏技巧

## 交互功能

- **导航菜单** - 点击导航链接平滑滚动到对应章节
- **展开/折叠** - 点击卡片展开详细内容
- **麻将牌交互** - 悬停和点击麻将牌查看效果
- **进度条** - 顶部显示阅读进度
- **回到顶部** - 滚动时显示回到顶部按钮
- **动画效果** - 元素进入视口时自动播放动画

## 技术栈
